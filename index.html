Top 10 Useful HTML Tags

<div>
The generic container for flow content, essential for grouping elements for styling.
<a>
The anchor tag, used for creating hyperlinks to other web pages or locations within the same page.
<img>
Embeds an image into the document.
<p>
Represents a paragraph of text.
<ul> / <ol>
Creates unordered and ordered lists.
<li>
Represents an item in a list.
<span>
A generic inline container for phrasing content, used for styling parts of text.
<input>
Used to create interactive controls for web-based forms.
<form>
Represents a document section containing interactive controls for submitting information.
<head>
Container for metadata (data about data) and is placed between the html tag and the body tag.

SEO & Header Tags

Search Engine Optimization (SEO) improves a website's visibility. Using correct header tags is crucial for search engines to understand the hierarchy and importance of content.

Important Header Tags for SEO:

  • <title>: technically in the head, but the most important tag for SEO. Defines the title in search results.
  • <h1>: The main heading of the page. Should contain the primary keyword and describe the page content. Usually only one per page.
  • <h2>: Level 2 headings. Break up content into main sections.
  • <h3>: Level 3 headings. Sub-sections under h2s.
  • <meta name="description">: While not a visible header, this meta tag provides the summary snippet in search results.

Font Sizes & Units

  • This text is 16px. Use 'px' for precise, fixed-size control, often for borders or base settings, though less scalable for accessibility.
  • This text is 1.2em. Use 'em' for sizing relative to the parent element's font size. Good for component-level scalability.
  • This text is 1.5rem. Use 'rem' (root em) for sizing relative to the root html font size. Best for global consistency and accessibility.
  • This text is 150%. Use '%' for fluid typography primarily used in responsive layouts.
  • This text is 12pt. Use 'pt' (points) primarily for print media stylesheets. Rarely used for screen design.
  • This text is 3vw. Use 'vw' (viewport width) for text that needs to scale dynamically with the browser window width, like hero headlines.

CSS Positioning Creative Area

Examples of static, relative, absolute, and sticky positioning.

Static

Default behavior. Use for standard flow content.

Relative

Offset from its normal position. Use when you need to nudge an element or create a context for an absolute child.

Absolute

Removed from flow, positioned relative to nearest positioned ancestor. Use for overlays, badges, or precise placement.

Sticky

Toggles between relative and fixed based on scroll. Use for headers that should stick to top.
